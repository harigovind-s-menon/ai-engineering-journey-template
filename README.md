# AI Engineering Journey — starter template

This repo is generated from a template when you connect GitHub on the tracker.
Each lesson has its own folder with a `notes.md`. Do the lesson, then mark it
complete by adding a `DONE.md` in that lesson's folder and pushing:

```
touch <lesson-folder>/DONE.md && git add -A && git commit -m "done: <lesson-id>" && git push
```

Your progress page reads this public repo and flips a lesson to ✓ once its
`DONE.md` exists. Nothing else to configure.
